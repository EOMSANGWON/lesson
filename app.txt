**git 가이드북**

[1] git 초기 세팅
    1. git 설치 (윈도우와 맥 버전이 따로 있음)
    2. git을 실행할 폴더 생성, 폴더 우클릭 후 터미널, 
        git config --global user.email "um4own@gmail.com"
        git config --global user.name "엄상원"
    후 빠져나옴

[2] git 실행
    1. 에디터실행, 작업폴더 위치에서 터미널 실행, 
        git init
    을 실행하면 git이 감시를 시작한다. (+repository를 생성)
    2. 작업 후 저장하기 위해서는 
        git add 파일명
        git commit -m '메모'
    를 터미널에 순서대로 실행시킨다. 기록 완료
    이후 코드를 수정하면 다시 add와 commit을 연달아 실행시킨다.
    ? add와 commit는 뭐가 다른거지?
